# ReWear

Welcome to the ReWear repository. This project follows a professional, scalable domain-driven architecture to separate concerns cleanly between client applications, server logic, and project-level information.

## Project Domains

✅ **[Frontend](./Frontend)** — Implemented
The complete React + Vite application featuring a modern glassmorphism UI, Framer Motion cinematic animations, and responsive layouts.

🚧 **[Backend](./Backend)** — Planned
Currently contains placeholder documentation outlining the future architecture (APIs, Database, Auth).

📚 **[Documentation](./Documentation)** — Available
Centralized project information, architecture diagrams, contribution guidelines, and setup instructions.

## Quick Start

To run the application locally, navigate into the Frontend domain:

```bash
cd Frontend
npm install
npm run dev
```
